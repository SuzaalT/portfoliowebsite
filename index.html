FLAPPYBIRD

This project is a clone of the popular mobile game Flappy Bird, originally developed by Dong Nguyen. The game features a simple yet addictive gameplay mechanic where players control a small bird by tapping the screen to make it flap its wings. The objective is to navigate through a series of pipes without colliding, creating a challenging and engaging experience.

LALIGURANS

This project involves creating a dynamic website for a restaurant in Toronto, aimed at enhancing its online presence and attracting more customers. The website will feature a user-friendly interface that showcases the restaurant's menu, ambiance, and unique offerings. Key elements will include an interactive menu section with descriptions and prices, an image gallery highlighting the restaurant's interior and dishes, and a reservation system to streamline booking. Utilizing HTML, CSS, and JavaScript, I aim to develop a responsive design that ensures an optimal viewing experience on both desktop and mobile devices. This project emphasizes my skills in front-end development while providing a platform for the restaurant to connect with its clientele and grow its business.
